# 📘 Assignment Submission Usage

## Guidelines
This document outlines the requirements and best practices for submitting assignments/projects. Please follow the instructions carefully to ensure smooth review and grading.

### 🔗 Preferred Submission Format
- **Primary: GitHub Repository link**
  - Branch must be `main`
  - Repository must be public / open‑source
  - Repository size should be within GitHub limits and fully downloadable
- **Alternative: Google Drive link**
  - Must be publicly accessible / not protected
  - Folder/file should be downloadable
  - File size should not exceed Google Drive limits (ideally < 1 GB)

### 📂 Project Structure
- Submit only the basic application with all required assignment and project code files.
- Do not include unnecessary files such as:
  - `node_modules` or equivalent dependency folders
  - `.env` or other sensitive configuration files
  - Build artifacts (`dist/`, `.next/`, `out/`, etc.)
  - Temporary/editor‑specific files (`.vscode/`, `.idea/`, etc.)

### 📦 Packages & Dependencies
- ❌ No extra modules or package files should be added.
- ✅ Use only what is strictly required for the assignment/project.
- Keep dependencies minimal and native whenever possible.

### ✅ Checklist Before Submission
- Ensure the app runs without errors.
- Code files are properly structured and named.
- No unnecessary/temporary files committed.
- Branch name is set to `main`.
- Repository or Google Drive link is publicly accessible and downloadable.
- Submission size is within GitHub / Google Drive limits.
- Proper documentation or comments where necessary.

### 📤 How to Submit
1. **Push your code to a GitHub repository.**
   - Branch: `main`
   - Public / open‑source
   - Downloadable and size within GitHub limits
   - Example: `https://github.com/username/project-name`
2. **If using Google Drive, ensure the folder/file is:**
   - Publicly accessible
   - Downloadable
   - Within Google Drive size limits (ideally < 1 GB)
   - Example: `https://drive.google.com/drive/folders/...`
3. **Share the link** as per the assignment submission instructions.
